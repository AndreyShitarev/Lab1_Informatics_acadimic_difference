# Отчёт по первой лабораторной работе 
**Задача:** 
Создать bash-скрипт, который при запуске должен выводить строку Welcome, и имя, задаваемое пользователем через терминал имя. 
## Шаги, проделанные для выполнения задачи:
### 1. Создание директории для работы
<img width="794" height="62" alt="DirMake" src="https://github.com/user-attachments/assets/873858db-051d-439a-a4e6-85861b0cf637" />

Команда `mkdir` создаёт директорию в указанном месте `/ITMO/academicDifference/Informatics/lab1`. Так как у меня не было директорий, предшествующих lab1, я использую флаг `-p`, который создаёт эти промежуточные каталоги, если их ещё нет. Это позволяет записать одну команду, вместого того, чтобы писать её 4 раза для каждой папки.
